1. 本分冊の役割

市場・認証編の役割は、単に国別情報を並べることではなく、 市場要求を設計要求へ翻訳するための前提条件を整理することにある。 同じレンジフードやガスコンロでも、住宅構造、調理文化、ガス種、電圧、 認証制度、販路の違いによって「売れる仕様」は大きく変わる。

1-1 市場情報を設計へつなぐ

住宅事情、施工制約、調理文化、設置慣行を把握することで、風量・静圧・火力・サイズ・騒音・安全機能などの設計条件が定まる。

1-2 国別比較だけでは足りない

商品別の競争地図と主要3社の訴求・代表仕様・販路まで見る必要がある。「その国で何が売れているか」が設計の現実解になる。

1-3 商品別競争地図の把握

レンジフード、ガスコンロ、IH、給湯器、食洗機の5商品について、各市場の主要ブランドと勝ち筋を一覧で整理する。

2. 市場比較の共通フレーム

各市場を比較する際は、以下の6軸で見ると情報が設計・営業・品質の全てに接続しやすい。

2-1 住宅事情：戸建 / マンション / 高層住宅 / HDB / Condo / 狭小キッチンなど。ダクト長・据付寸法・搬入性に直結する。
2-2 調理・入浴・家事文化：高火力炒め物の比率、揚げ物頻度、IH普及、浴室サイズ、食洗機の定着度など。必要機能が変わる。
2-3 電圧・周波数・ガス種：100V / 110V / 220V / 230V / 240V、50Hz / 60Hz、LPG / PNG / Towngas / 13A など。電装・燃焼設計の基礎条件。
2-4 施工・据付制約：共有ダクト、外壁排気、HDB制約、浴室配管、開口寸法、電気・ガス工事の慣行など。施工性と保証に関わる。
2-5 認証・法規・表示：PSE、CCC、BSMI、Safety Mark、SIRIM、EMSD等。販売可能性と開発リードタイムを左右する。
2-6 販売チャネル：量販、EC、施工店、代理店、デベロッパー案件、B2B / B2Cの違い。商品仕様と価格帯の設計前提になる。

3. 日本市場 🇯🇵

静音 / 清掃性 / 保守性 / 国内品質 | PSE | 100V・200V / 50-60Hz

品質・信頼性・静音・清掃性・保守性を重視する成熟市場。 国内ブランドの信頼が強く、施工性・アフターサービスも購買要因になる。 キッチン空間は比較的コンパクトで、60/75/90cm幅の据付が一般的。

🔥 ガス：13A 都市ガス / LPG

🏠 住宅：戸建・マンション混在

4. 中国市場 🇨🇳

高風量 / 高火力 / B2B二重構造 | CCC / GB | 220V / 50Hz

最大規模市場であり、現地ブランドの競争が最も激しい。 レンジフード・ガスコンロ・総合家電の各カテゴリで、 高風量・高火力・スマート機能・派手な訴求が強い。

5. 台湾市場 🇹🇼

110V / 60Hz | 省エネ / IoT / 技術訴求 | BSMI / CNS

技術仕様への理解が比較的高く、日本・欧州ブランドへの信頼も強い市場。 110V / 60Hz という電源条件が大きな特徴で、他市場との共通化を難しくする。

電源上の注意

110V / 60Hz対応は、他の220〜240V市場と回路・モータ・熱設計の共通化を難しくする。専用設計または広電圧対応の検討が必要。Sakuraのように複数カテゴリ横断で統一体験を作る戦略が強い。

9. 競合ベンチマークの読み解き方

9-1 ブランドポジション：高級 / 普及 / 専業 / 総合 / 施工店主導 / EC主導を見分ける。
9-2 主訴求：吸引、火力、省エネ、清掃、静音、IoT、抗菌など、何を最も前面に出しているかを見る。
9-3 代表仕様：m³/h、Pa、dB、kW、L/min、place settingsなどの数値で比較する。
9-4 販売チャネル：量販・EC・施工店・B2B案件のどこが強いかで、商品企画が変わる。
9-5 住宅事情との関係：高静圧が必要なのか、薄型が必要なのか、再循環が必要なのかを住宅条件と結びつけて読む。
読み解き視点何を見るか設計にどうつながるか
ブランド専業か総合か、高級か普及か競合の勝ち筋と自社の立ち位置整理
主訴求広告・商品説明で繰り返す言葉市場が求める価値の仮説化
数値仕様風量、静圧、火力、温度制御、容量等最低仕様と差別化仕様の設定
住宅制約HDB、高層住宅、共同ダクト、狭小キッチン寸法・静圧・設置方式の決定
認証強制認証、表示、エネルギーラベル開発日程・部材選定・試験計画へ反映

12. 市場・認証編まとめ

12-1 どの市場で何を重視すべきか

中国は高風量・高火力・B2B二重構造、台湾は110Vと細かな機能訴求、シンガポールはHDBと再循環、マレーシアは耐湿と価格バランス、香港は狭小高層、日本は静音・清掃・保守。

12-2 どの商品で何が勝ち筋か

レンジフードは市場ごとの空間制約と油煙量対応、ガスは火力と安全の両立、IHは制御・連動・UI、給湯器は設置方式と熱源、食洗機はサイズと乾燥方式が勝負になる。
