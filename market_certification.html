1. 本分冊の役割

市場・認証編の役割は、単に国別情報を並べることではなく、 市場要求を設計要求へ翻訳するための前提条件を整理することにある。 同じレンジフードやガスコンロでも、住宅構造、調理文化、ガス種、電圧、 認証制度、販路の違いによって「売れる仕様」は大きく変わる。

1-1 市場情報を設計へつなぐ

住宅事情、施工制約、調理文化、設置慣行を把握することで、 風量・静圧・火力・サイズ・騒音・安全機能などの設計条件が定まる。

1-2 国別比較だけでは足りない

市場編では、商品別の競争地図と主要3社の訴求・代表仕様・販路まで見る必要がある。 「その国で何が売れているか」が設計の現実解になる。

1-3 商品別競争地図の把握

レンジフード、ガスコンロ、IH、給湯器、食洗機の5商品について、 各市場の主要ブランドと勝ち筋を一覧で整理する。

2. 市場比較の共通フレーム

各市場を比較する際は、以下の6軸で見ると情報が設計・営業・品質の全てに接続しやすい。

2-1 住宅事情

戸建 / マンション / 高層住宅 / HDB / Condo / 狭小キッチンなど。ダクト長・据付寸法・搬入性に直結する。

2-2 調理・入浴・家事文化

高火力炒め物の比率、揚げ物頻度、IH普及、浴室サイズ、食洗機の定着度など。必要機能が変わる。

2-3 電圧・周波数・ガス種

100V / 110V / 220V / 230V / 240V、50Hz / 60Hz、LPG / PNG / Towngas / 13A など。電装・燃焼設計の基礎条件。

2-4 施工・据付制約

共有ダクト、外壁排気、HDB制約、浴室配管、開口寸法、電気工事・ガス工事の慣行など。施工性と保証に関わる。

2-5 認証・法規・表示

PSE、CCC、BSMI、Safety Mark、SIRIM、EMSD 等。販売可能性と開発リードタイムを左右する。

2-6 販売チャネル

量販、EC、施工店、代理店、デベロッパー案件、B2B / B2C の違い。商品仕様と価格帯の設計前提になる。

3. 日本市場

3-1 市場概況

品質・信頼性・静音・清掃性・保守性を重視する成熟市場。 国内ブランドの信頼が強く、施工性・アフターサービスも購買要因になる。 キッチン空間は比較的コンパクトで、60/75/90cm 幅の据付が一般的。

日本市場から学ぶ点

高機能だけでなく、静音・清掃・保証・部材品質・施工説明の完成度まで含めて商品力が評価される。

4. 中国市場

4-1 市場概況

最大規模市場であり、現地ブランドの競争が最も激しい。 レンジフード・ガスコンロ・総合家電の各カテゴリで、 高風量・高火力・スマート機能・派手な訴求が強い。

4-2 毛坯房 / 精装修

中国市場は、個人施主向けの毛坯房と、デベロッパー主導の精装修で要求が分かれる。 毛坯房では最大風量・高火力・目立つ機能訴求、 精装修では静音・安定性・共同ダクト適合・標準化が重要となる。

市場が大きい分、訴求力と仕様の両立が必要。毛坯房向けと精装修向けでは、同じ商品でも設計思想を分ける発想が重要。

台湾市場から学ぶ点

Sakuraのように複数カテゴリ横断で統一体験を作る戦略が強い。機能の細やかさとブランド一貫性が重要。

7. マレーシア市場

8. 香港市場

8-1 市場概況

  • ガス：Towngas / LPG
  • 住宅：高層・狭小・都市型
  • 重点：薄型、省スペース、静音、高温除菌
  • 認証：EMSD、電気製品安全規則

狭小でも妥協しない高機能・高品質の見せ方が重要。小さい空間での使い勝手と意匠性が競争軸になる。

9. 競合ベンチマークの読み解き方

9-1 ブランドポジション

高級 / 普及 / 専業 / 総合 / 施工店主導 / EC主導を見分ける。

9-2 主訴求

吸引、火力、省エネ、清掃、静音、IoT、抗菌など、何を最も前面に出しているかを見る。

9-3 代表仕様

m³/h、Pa、dB、kW、L/min、place settingsなどの数値で比較する。

9-4 販売チャネル

量販・EC・施工店・B2B案件のどこが強いかで、商品企画が変わる。

9-5 住宅事情との関係

高静圧が必要なのか、薄型が必要なのか、再循環が必要なのかを住宅条件と結びつけて読む。

読み解き視点 何を見るか 設計にどうつながるか

12. 市場・認証編まとめ

12-1 どの市場で何を重視すべきか

中国は高風量・高火力・B2B二重構造、台湾は110Vと細かな機能訴求、 シンガポールはHDBと再循環、マレーシアは耐湿と価格バランス、 香港は狭小高層、日本は静音・清掃・保守。

12-2 どの商品で何が勝ち筋か

レンジフードは市場ごとの空間制約と油煙量対応、ガスは火力と安全の両立、 IHは制御・連動・UI、給湯器は設置方式と熱源、食洗機はサイズと乾燥方式が勝負になる。
