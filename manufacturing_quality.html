1. 本分冊の位置づけ

製造・品質編の役割は、技術編で整理した機能・構造・材料を、実際に 量産できる状態へ落とし込むことである。 単に「工場工程を説明する」のではなく、競合仕様が要求する公差、 工程難度、検査条件、供給者品質、保守性までを含めて扱う。

製造視点：高機能な設計でも、量産時にばらつき・組立負荷・歩留まり悪化を招けば商品として成立しない。
品質視点：不具合の未然防止、工程内検出、出荷前保証、市場不具合の戻し込みまで一貫して見る必要がある。
競合起点：薄型筐体、黒ガラス、親水コート、Flex機能、Zeolith乾燥などは製造難所を伴う。差別化と量産性を同時に見る。
分冊間の接続：市場編の要求と技術編の機能分解を、ここで工程・検査・品質KPIへ変換する。

2. 開発から量産までの全体像

量産品質は製造現場だけで決まるのではなく、商品企画から市場品質までの連鎖で作られる。

RPN = S（重大度）× O（発生頻度）× D（検出難度）

商品企画対象市場、価格帯、顧客価値、競合比較目標原価、供給可能部材、工程難度の初期見積り
設計構造、材料、回路、制御、認証対応DFM / DFA、公差、サービス性、試験性
試作試作品製作、構造・性能・認証予備確認加工性確認、初期不具合抽出、工程仮説検証
評価性能試験、耐久試験、環境試験、認証試験検査基準、故障モード、量産ばらつき余裕の確認
量産立上げ治具・金型・工程設計、作業標準、教育Cpk、工程内検査、初期流動管理、ライン能力
市場品質出荷後の故障・クレーム・サービス再発防止、設計変更、供給者是正、検査見直し

3. 共通DFM・DFAの基本

住宅設備機器の量産性は、部品点数、組立方向、加工ルール、公差設計、サービス性で決まる。 単一部品最適ではなく全体組立最適が重要になる。

コスト・工数・品質：加工歩留まり、組立時間、手直し率、保証費を含めて見る。安い構造が必ずしも総コスト最適ではない。
部品点数低減：部品数が減ると、購買点数、在庫、誤組立、締結点数、工程時間を同時に削減できる。
組立性：一方向組立、姿勢変更の少なさ、工具アクセス、ポカヨケ、同一トルク条件が工程安定に効く。
サービス性：消耗品交換、清掃、補修部品交換のしやすさは、市場品質とアフターコストに直結する。
設計項目基本ルール狙い
穴径一般に穴径 ≥ 板厚 t打抜き変形・パンチ折損・バリ悪化の回避
穴-端面距離一般に 1.5t〜2t 以上端部割れ・変形防止
穴-曲げ距離2t + R 以上が目安曲げ部変形と穴位置ズレ防止
最小曲げR材料に応じて 0.5t〜1.5t 以上割れ・スプリングバック過大防止
ねじ締結トルク条件統一、工具アクセス確保締め忘れ・舐め・オーバートルク防止
ポカヨケ左右非対称、異形コネクタ、色分け誤組立防止

4. レンジフードの製造工程と品質

外観と性能が同時に問われる製品。薄型化・清掃性・静音化を競争軸にすると、板金精度、ケーシング組付、流路面品質の要求が上がる。

工程主な内容品質重点
板金ブランキング、NCT、レーザー、曲げ反り、バリ、穴位置、外観キズ
溶接 / 接合スポット、MIG、リベット、カシメ変形、強度、密着、異音要因
塗装・表面処理化成処理、粉体塗装、SUS仕上げ膜厚、色差、密着、防錆
ファン・モータ組付ケーシング、シロッコ、モータ、制御回転バランス、騒音、振動、配線
フィルタ・整流板着脱部、ロック部、オイルトレイガタ、着脱力、密着、清掃性
最終組立・検査通電、風量、騒音、外観、梱包仕様値達成、出荷不良防止
大型板金の難所：薄板大面積は反りやすく、外観にも直結する。薄型筐体ほど板厚・補強リブ・フランジ設計が重要。
騒音・振動：ファン単体ではなく、モータ取付剛性、ケーシング、共振、ダクト接続まで含めて評価する必要がある。
清掃構造：整流板の保持精度、フィルタの面圧、オイルトレイの排油角度が使い勝手に効く。
競合構造の難度：Auto Fan Clean や撥油 / 親水コーティングは、通常板金フードより製造条件がシビアになる。
風量評価：JIS B 8330 準拠の測定系で確認 / 騒音評価：JIS C 9603 系または同等条件で dB(A) 管理

5. ガスコンロの製造工程と品質

火力・安全・意匠・気密の4要素が強く絡む製品。天板外観が商品価値を左右する一方、内部ではガス漏れ・点火・炎安定性・安全弁作動が最優先になる。

工程主な内容品質重点
天板加工ガラス / ステンレス / エナメル外装形成割れ、欠け、表面傷、寸法精度
バーナー組立炎口、混合部、キャップ、熱電対炎安定、熱効率、点火性
ガス系組立配管、弁、ノズル、シール漏れ、締結、ガス種適合
点火・安全機構圧電、イグナイタ、FFD、過熱防止点火率、立消え安全、応答時間
最終燃焼評価火力、炎色、偏炎、消火挙動燃焼安定、CO抑制、再現性
ガラストップの難所：高級感は高いが、端部欠け、熱衝撃、組付応力、輸送破損への対策が必要。
鋳鉄五徳：重量・見た目・耐熱は強いが、塗装密着、錆、寸法ばらつき、ガタ管理が課題。
気密・漏れ：工程内での漏れ検査、締結トルク管理、シール部材管理が最重要の品質ゲートになる。
温調機能：センサ位置、公差、応答性、ソフト補正がずれると、競合水準の温調精度に届かない。

6. IH / 電気ホブの製造工程と品質

機械系よりも電装・熱・EMC・ソフトの影響が大きい。ガラスとコイルの位置関係、放熱、タッチ感度、ノイズ対策が品質を左右する。

工程主な内容品質重点
ガラス天板強化ガラス / ガラスセラミックの受入・貼合割れ、反り、外観、耐熱衝撃
コイル組立コイル位置決め、接着、絶縁発熱効率、位置ズレ、絶縁安全
パワー基板実装IGBT / MOSFET、PFC、制御回路半田品質、発熱、電気安全、EMC
冷却系組立ファン、ヒートシンク、風路温度上昇、騒音、寿命
UI / 表示部タッチセンサ、表示、ロジック誤動作、感度、視認性、防水性
EMC難度：高周波スイッチングとタッチUI、通信モジュールが近接するため、競合高機能機ほど設計・評価の難度が上がる。
冷却：PowerBoost や FlexInduction 系は発熱密度が高く、熱拡散設計とファン制御が商品寿命に直結する。
ガラス天板：見た目と清掃性に優れるが、局所荷重・熱衝撃・組付応力に対するマージンが必要。
UI品質：水滴、油膜、蒸気、手袋、外来ノイズ下でのタッチ安定性は市場クレームになりやすい。

7. 給湯器の製造工程と品質

水・熱・圧力・電気・燃焼が絡むため、構造的な安全率と長期耐久性の確保が特に重要。instantとstorageでは工程そのものが大きく異なる。

方式主工程品質重点
瞬間式熱交換器、燃焼部、流量制御、電装温度安定、燃焼安全、流量制御、スケール対策
貯湯式タンク溶接、内面処理、断熱材充填、ヒータ組付耐圧、防食、断熱、衛生、漏水
ハイブリッドヒートポンプ + 補助熱源 + 制御統合統合制御、効率、設置条件、部品点数増による信頼性
タンク品質：内面エナメル、SUS選定、Mgアノード、耐圧試験の出来で寿命が大きく変わる。
防水・漏水：継手、シール、溶接部、熱サイクルによる膨張差を考えた設計・検査が必要。
衛生：抗菌機能だけでなく、滞留部の少なさ、排水性、スケール・腐食管理が長期品質に効く。
保証設計：タンク系は保証期間が長く、設計時点でフィールド故障コストを見込む必要がある。

8. 食洗機の製造工程と品質

水密、洗浄性能、乾燥性能、静音、収納性が絡み合う製品。ビルトインと卓上では構造が大きく異なり、限られた容積内で流体・熱・騒音を最適化する必要がある。

工程主な内容品質重点
槽体組立ステンレス / 樹脂槽、シール、断熱材漏水、変形、耐熱、耐洗剤性
ポンプ・配管循環ポンプ、排水、スプレーアーム流量、騒音、漏れ、詰まり
ラック・扉収納部、レール、ヒンジ、ドアパッキン操作力、ガタ、密閉、耐久
ヒータ・乾燥系ヒーター、Zeolith、換気流路乾燥性能、温度安全、結露
制御・センサ濁度、温度、水位、コース制御自動制御精度、誤検知、再現性
漏水管理：部材精度、シール材、熱膨張差、輸送後組付ズレまで見ないと市場漏水不良になる。
乾燥方式：Zeolith や高度乾燥構造は強い差別化要素だが、構造・コスト・評価項目が増える。
静音：ポンプ振動、配管共振、扉剛性、槽体遮音、設置条件の全てが騒音に影響する。
小型化：卓上・portable型は差別化しやすい一方、容量・排水・乾燥・ユーザー期待値の調整が難しい。

10. 製品別FMEA・なぜなぜ分析の使い方

FMEAは未然防止、なぜなぜ分析は再発防止。両者は代替関係ではなく、役割が異なる。

RPN = Severity × Occurrence × Detection
商品FMEAで特に見る故障モードなぜなぜ分析の典型テーマ
レンジフード異音、風量不足、塗装剥離、フィルタ着脱不良なぜ騒音が出たか、なぜ油漏れしたか
ガスコンロガス漏れ、点火不良、偏炎、天板割れなぜ点火率が低いか、なぜガラスが割れたか
IH過熱、タッチ誤動作、EMC不適合、ガラス割れなぜ誤タッチしたか、なぜ高温停止したか
給湯器漏水、防食不良、温度不安定、安全弁不作動なぜ漏れたか、なぜ温度が揺れるか
食洗機漏水、乾燥不足、異音、排水不良なぜ洗えないか、なぜ乾かないか

FMEAのポイント

  • 故障モードを部品単位ではなく機能単位でも見る
  • 検出策が本当に工程で機能するか確認する
  • 対策後のRPN再評価を必ず行う

なぜなぜ分析のポイント

  • 人のミスで止めず、仕組み・条件・管理まで掘る
  • 再現条件を確認してから因果を決める
  • 是正処置と再発防止を分けて考える

11. 工程内品質と出荷品質

品質は最後の検査だけで作るものではない。受入から市場フィードバックまで複数のゲートを設定する必要がある。

段階主な管理内容典型指標
受入検査（IQC）外観、寸法、材料証明、重要特性の初期確認受入不良率、ロット承認率
工程監視（IPQC）トルク、電流、膜厚、漏れ、温度、圧力、寸法Cpk、工程不良率、再作業率
完成品検査（FQC）全機能、外観、通電、安全、梱包前確認初回合格率、最終不良率
出荷品質（OQC）ロット判定、AQL、ラベル・付属品・梱包出荷不良率、クレーム率
市場品質返品、故障、サービス情報の回収PPM、不具合件数、保証費率
受入：供給者品質が不安定な部品は、工程内で吸収しようとせず入口で止めるべき。
工程：トルク・漏れ・膜厚・電気特性のように、後工程で見つけにくいものは早い段階で監視する。
完成品：全モード確認と外観判定を標準化し、検査員依存を減らすことが重要。
市場戻し：クレームをサービス部門に閉じ込めず、設計・品質・購買・製造へ戻して構造的に潰す。

12. 競合ベンチマークを品質改善へつなぐ方法

「なぜその機能が成立しているか」を量産条件に翻訳して初めて意味を持つ。

競合差分製造で効く論点品質で効く論点社内で確認すべきこと
薄型レンジフード板金反り、モータ配置、組立順序異音、風量ばらつき補強設計、実機騒音、施工許容差
黒ガラストップ搬送治具、組付応力、端部保護欠け、割れ、外観傷輸送試験、外観基準、熱衝撃評価
高精度温調センサ位置、公差、基板校正温度誤差、再現性校正工程、工程能力、使用条件差
Zeolith乾燥部材追加、流路複雑化、組立増乾燥性能、安全温度長期耐久、コスト、サービス性

13. 製造・品質編まとめ

13-1 量産視点で見た勝ち筋

競合に勝つには高機能だけでは不十分で、その機能を歩留まり良く、安定品質で、適正コストで出せる量産設計が必要である。

13-2 技術編との接続

技術編で整理した機能・構造・材料を、ここで工程、部材、検査、品質保証へ具体的に翻訳した。
