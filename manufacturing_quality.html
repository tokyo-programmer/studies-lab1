1. 本分冊の位置づけ

製造・品質編の役割は、技術編で整理した機能・構造・材料を、 実際に量産できる状態へ落とし込むことである。 ここでは単に「工場工程を説明する」のではなく、 競合仕様が要求する公差、工程難度、検査条件、供給者品質、保守性までを含めて扱う。

1-1 製造視点

高機能な設計でも、量産時にばらつき・組立負荷・歩留まり悪化を招けば商品として成立しない。

1-2 品質視点

不具合の未然防止、工程内検出、出荷前保証、市場不具合の戻し込みまで一貫して見る必要がある。

1-3 競合起点

薄型筐体、黒ガラス、親水コート、Flex機能、Zeolith乾燥などは製造難所を伴う。差別化と量産性を同時に見る。

1-4 分冊間の接続

市場編の要求と技術編の機能分解を、ここで工程・検査・品質KPIへ変換する。

2. 開発から量産までの全体像

量産品質は製造現場だけで決まるのではなく、商品企画から市場品質までの連鎖で作られる。 特に住宅設備機器では、認証、施工性、アフターサービス、補修部品供給まで含めて量産準備と考える必要がある。

段階 主な内容 製造・品質の論点
商品企画 対象市場、価格帯、顧客価値、競合比較 目標原価、供給可能部材、工程難度の初期見積り
設計 構造、材料、回路、制御、認証対応 DFM / DFA、公差、サービス性、試験性
試作 試作品製作、構造・性能・認証予備確認 加工性確認、初期不具合抽出、工程仮説検証
評価 性能試験、耐久試験、環境試験、認証試験 検査基準、故障モード、量産ばらつき余裕の確認
量産立上げ 治具・金型・工程設計、作業標準、教育 Cpk、工程内検査、初期流動管理、ライン能力
市場品質 出荷後の故障・クレーム・サービス 再発防止、設計変更、供給者是正、検査見直し
RPN = S × O × D

故障モードの優先順位付けには FMEA の RPN を用いるが、 実務上は重大度が高い項目を優先し、単純な数値の大小だけで判断しないことが重要である。

3. 共通DFM・DFAの基本

住宅設備機器の量産性は、部品点数、組立方向、加工ルール、公差設計、サービス性で決まる。 特に板金・樹脂・ガラス・電装をまたぐ商品では、単一部品最適ではなく全体組立最適が重要になる。

3-1 コスト・工数・品質

安い構造が必ずしも総コスト最適ではない。加工歩留まり、組立時間、手直し率、保証費を含めて見る。

3-2 部品点数低減

部品数が減ると、購買点数、在庫、誤組立、締結点数、工程時間を同時に削減できる。

3-3 組立性

一方向組立、姿勢変更の少なさ、工具アクセス、ポカヨケ、同一トルク条件が工程安定に効く。

3-4 サービス性

消耗品交換、清掃、補修部品交換のしやすさは、市場品質とアフターコストに直結する。

設計項目 基本ルール 狙い
穴径 一般に穴径 ≥ 板厚 t 打抜き変形・パンチ折損・バリ悪化の回避
穴-端面距離 一般に 1.5t〜2t 以上 端部割れ・変形防止
穴-曲げ距離 2t + R 以上が目安 曲げ部変形と穴位置ズレ防止
最小曲げR 材料に応じて 0.5t〜1.5t 以上 割れ・スプリングバック過大防止
ねじ締結 トルク条件統一、工具アクセス確保 締め忘れ・舐め・オーバートルク防止
ポカヨケ 左右非対称、異形コネクタ、色分け 誤組立防止

4. レンジフードの製造工程と品質

4-1 代表工程

レンジフードは大型板金、ファン・モータ、フィルタ、塗装外観、電装を組み合わせる製品であり、 外観と性能が同時に問われる。特に薄型化・清掃性・静音化を競争軸にすると、 板金精度、ケーシング組付、流路面品質の要求が上がる。

工程 主な内容 品質重点
板金 ブランキング、NCT、レーザー、曲げ 反り、バリ、穴位置、外観キズ
溶接 / 接合 スポット、MIG、リベット、カシメ 変形、強度、密着、異音要因
塗装・表面処理 化成処理、粉体塗装、SUS仕上げ 膜厚、色差、密着、防錆
ファン・モータ組付 ケーシング、シロッコ、モータ、制御 回転バランス、騒音、振動、配線
フィルタ・整流板 着脱部、ロック部、オイルトレイ ガタ、着脱力、密着、清掃性
最終組立・検査 通電、風量、騒音、外観、梱包 仕様値達成、出荷不良防止

大型板金の難所

薄板大面積は反りやすく、外観にも直結する。薄型筐体ほど板厚・補強リブ・フランジ設計が重要。

騒音・振動

ファン単体ではなく、モータ取付剛性、ケーシング、共振、ダクト接続まで含めて評価する必要がある。

清掃構造

整流板の保持精度、フィルタの面圧、オイルトレイの排油角度が使い勝手に効く。

競合構造の難度

Auto Fan Clean や撥油 / 親水コーティングは、通常板金フードより製造条件がシビアになる。

風量評価：JIS B 8330 準拠の測定系で確認
騒音評価：JIS C 9603 系または同等条件で dB(A) 管理

5. ガスコンロの製造工程と品質

ガスコンロは、火力・安全・意匠・気密の4要素が強く絡む製品である。 天板外観が商品価値を左右する一方、内部ではガス漏れ・点火・炎安定性・安全弁作動が最優先になる。

工程 主な内容 品質重点
天板加工 ガラス / ステンレス / エナメル外装形成 割れ、欠け、表面傷、寸法精度
バーナー組立 炎口、混合部、キャップ、熱電対 炎安定、熱効率、点火性
ガス系組立 配管、弁、ノズル、シール 漏れ、締結、ガス種適合
点火・安全機構 圧電、イグナイタ、FFD、過熱防止 点火率、立消え安全、応答時間
最終燃焼評価 火力、炎色、偏炎、消火挙動 燃焼安定、CO抑制、再現性

ガラストップの難所

高級感は高いが、端部欠け、熱衝撃、組付応力、輸送破損への対策が必要。

鋳鉄五徳

重量・見た目・耐熱は強いが、塗装密着、錆、寸法ばらつき、ガタ管理が課題。

気密・漏れ

工程内での漏れ検査、締結トルク管理、シール部材管理が最重要の品質ゲートになる。

温調機能

センサ位置、公差、応答性、ソフト補正がずれると、競合水準の温調精度に届かない。

6. IH / 電気ホブの製造工程と品質

IHは、加熱コイル・電力変換回路・ガラス天板・冷却・UIが一体化した製品であり、 機械系よりも電装・熱・EMC・ソフトの影響が大きい。 特にガラスとコイルの位置関係、放熱、タッチ感度、ノイズ対策が品質を左右する。

工程 主な内容 品質重点
ガラス天板 強化ガラス / ガラスセラミックの受入・貼合 割れ、反り、外観、耐熱衝撃
コイル組立 コイル位置決め、接着、絶縁 発熱効率、位置ズレ、絶縁安全
パワー基板実装 IGBT / MOSFET、PFC、制御回路 半田品質、発熱、電気安全、EMC
冷却系組立 ファン、ヒートシンク、風路 温度上昇、騒音、寿命
UI / 表示部 タッチセンサ、表示、ロジック 誤動作、感度、視認性、防水性
主要評価：温度上昇 / EMC / ノイズ / 火力制御再現性 / 残熱表示 / センサ誤動作

ガラス天板

見た目と清掃性に優れるが、局所荷重・熱衝撃・組付応力に対するマージンが必要。

EMC難度

高周波スイッチングとタッチUI、通信モジュールが近接するため、競合高機能機ほど設計・評価の難度が上がる。

冷却

PowerBoost や FlexInduction 系は発熱密度が高く、熱拡散設計とファン制御が商品寿命に直結する。

UI品質

水滴、油膜、蒸気、手袋、外来ノイズ下でのタッチ安定性は市場クレームになりやすい。

7. 給湯器の製造工程と品質

給湯器は、水・熱・圧力・電気・燃焼が絡むため、 構造的な安全率と長期耐久性の確保が特に重要な商品である。 instant と storage では工程そのものが大きく異なり、 タンク製品では防食・断熱・溶接品質、瞬間式では熱交換・流量制御・燃焼安定が鍵となる。

方式 主工程 品質重点
瞬間式 熱交換器、燃焼部、流量制御、電装 温度安定、燃焼安全、流量制御、スケール対策
貯湯式 タンク溶接、内面処理、断熱材充填、ヒータ組付 耐圧、防食、断熱、衛生、漏水
ハイブリッド ヒートポンプ + 補助熱源 + 制御統合 統合制御、効率、設置条件、部品点数増による信頼性

タンク品質

内面エナメル、SUS選定、Mgアノード、耐圧試験の出来で寿命が大きく変わる。

防水・漏水

継手、シール、溶接部、熱サイクルによる膨張差を考えた設計・検査が必要。

衛生

抗菌機能だけでなく、滞留部の少なさ、排水性、スケール・腐食管理が長期品質に効く。

保証設計

タンク系は保証期間が長く、設計時点でフィールド故障コストを見込む必要がある。

8. 食洗機の製造工程と品質

食洗機は、水密、洗浄性能、乾燥性能、静音、収納性が絡み合う製品である。 ビルトインと卓上では構造が大きく異なり、特に都市市場向けの小型・Drawer型・卓上型では、 限られた容積内で流体・熱・騒音を最適化する必要がある。

工程 主な内容 品質重点
槽体組立 ステンレス / 樹脂槽、シール、断熱材 漏水、変形、耐熱、耐洗剤性
ポンプ・配管 循環ポンプ、排水、スプレーアーム 流量、騒音、漏れ、詰まり
ラック・扉 収納部、レール、ヒンジ、ドアパッキン 操作力、ガタ、密閉、耐久
ヒータ・乾燥系 ヒーター、Zeolith、換気流路 乾燥性能、温度安全、結露
制御・センサ 濁度、温度、水位、コース制御 自動制御精度、誤検知、再現性

漏水管理

部材精度、シール材、熱膨張差、輸送後組付ズレまで見ないと市場漏水不良になる。

乾燥方式

Zeolith や高度乾燥構造は強い差別化要素だが、構造・コスト・評価項目が増える。

静音

ポンプ振動、配管共振、扉剛性、槽体遮音、設置条件の全てが騒音に影響する。

小型化

卓上・portable型は差別化しやすい一方、容量・排水・乾燥・ユーザー期待値の調整が難しい。

9. 材料・表面処理・部品品質

競合差の多くは、意匠そのものよりも、 どの材料を使い、どう表面処理し、どこまで供給者品質を作り込んでいるかに現れる。 とくに黒ガラス、SUS、鋳鉄、樹脂、コーティング、断熱材、防食材は全商品横断で重要である。

材料 / 処理 主な用途 利点 品質上の注意
SUS304 / SUS430 フード外装、給湯器、槽体 耐食、意匠 加工硬化、表面傷、材料差
黒ガラス / ガラスセラミック ガス天板、IH天板、UI 高級感、清掃性 割れ、欠け、熱衝撃、輸送保護
粉体塗装 板金外装 防錆、意匠、環境性 膜厚、色差、密着、ファラデー部の塗り不足
撥油 / 親水 / 防汚コート フード、ガラストップ、内部流路 清掃性向上 洗剤耐久、摩耗、量産ばらつき
断熱材 給湯器、食洗機、IH 熱効率、静音 吸湿、劣化、施工性、厚み制約
樹脂（PP / PA / PC / ABS / PPS） 操作部、カバー、トレイ、ホルダ 軽量、成形自由度 ESC、耐熱、耐洗剤、難燃、吸湿

供給者品質管理

材料証明、RoHS / REACH、寸法管理、初物承認、変更管理、定期監査をセットで運用する必要がある。

外観標準

A面・B面・C面で許容基準を分ける。高級市場ではA面の傷・異物・段差基準をより厳しく設定する。

変更管理

部材変更は見た目だけでなく、認証・熱・電気安全・洗剤耐久に影響するため、ECR/ECO管理が必須。

10. 製品別FMEA・なぜなぜ分析の使い方

FMEA は設計と工程の両方で実施し、量産前に潜在不具合を潰すための道具である。 一方、なぜなぜ分析は市場や工程で実際に起きた不具合の根本原因を掘るために使う。 両者は代替関係ではなく、未然防止と再発防止で役割が異なる。

RPN = Severity × Occurrence × Detection
商品 FMEAで特に見る故障モード なぜなぜ分析の典型テーマ
レンジフード 異音、風量不足、塗装剥離、フィルタ着脱不良 なぜ騒音が出たか、なぜ油漏れしたか
ガスコンロ ガス漏れ、点火不良、偏炎、天板割れ なぜ点火率が低いか、なぜガラスが割れたか
IH 過熱、タッチ誤動作、EMC不適合、ガラス割れ なぜ誤タッチしたか、なぜ高温停止したか
給湯器 漏水、防食不良、温度不安定、安全弁不作動 なぜ漏れたか、なぜ温度が揺れるか
食洗機 漏水、乾燥不足、異音、排水不良 なぜ洗えないか、なぜ乾かないか

FMEAのポイント

  • 故障モードを部品単位ではなく機能単位でも見る
  • 検出策が本当に工程で機能するか確認する
  • 対策後のRPN再評価を必ず行う

なぜなぜ分析のポイント

  • 人のミスで止めず、仕組み・条件・管理まで掘る
  • 再現条件を確認してから因果を決める
  • 是正処置と再発防止を分けて考える

11. 工程内品質と出荷品質

品質は最後の検査だけで作るものではない。 受入、工程監視、完成品検査、梱包、出荷判定、市場フィードバックの各段階で、 不良流出を止める複数のゲートを設定する必要がある。

段階 主な管理内容 典型指標
受入検査 外観、寸法、材料証明、重要特性の初期確認 受入不良率、ロット承認率
工程監視 トルク、電流、膜厚、漏れ、温度、圧力、寸法 Cpk、工程不良率、再作業率
完成品検査 全機能、外観、通電、安全、梱包前確認 初回合格率、最終不良率
出荷品質 ロット判定、AQL、ラベル・付属品・梱包 出荷不良率、クレーム率
市場品質 返品、故障、サービス情報の回収 PPM、不具合件数、保証費率

受入

供給者品質が不安定な部品は、工程内で吸収しようとせず入口で止めるべき。

工程

トルク・漏れ・膜厚・電気特性のように、後工程で見つけにくいものは早い段階で監視する。

完成品

全モード確認と外観判定を標準化し、検査員依存を減らすことが重要。

市場戻し

クレームをサービス部門に閉じ込めず、設計・品質・購買・製造へ戻して構造的に潰す。

12. 競合ベンチマークを品質改善へつなぐ方法

競合ベンチマークは、単なるカタログ比較ではなく、 「なぜその機能が成立しているか」を量産条件に翻訳して初めて意味を持つ。 高級感のある黒ガラス、高捕集率、低騒音、高精度温調、Zeolith乾燥などは、 全て製造・品質特性の裏付けを必要とする。

競合差分 製造で効く論点 品質で効く論点 社内で確認すべきこと
薄型レンジフード 板金反り、モータ配置、組立順序 異音、風量ばらつき 補強設計、実機騒音、施工許容差
黒ガラストップ 搬送治具、組付応力、端部保護 欠け、割れ、外観傷 輸送試験、外観基準、熱衝撃評価
高精度温調 センサ位置、公差、基板校正 温度誤差、再現性 校正工程、工程能力、使用条件差
Zeolith乾燥 部材追加、流路複雑化、組立増 乾燥性能、安全温度 長期耐久、コスト、サービス性
親水 / 撥油コート 前処理、塗布条件、膜厚管理 耐洗剤性、摩耗耐久 市場洗剤との相性試験、寿命試験

どの差分が製造で効くか

構造・部材・公差・組立点数・治具の難易度に落とし込む。

どの差分が調達で効くか

特殊材料、ガラス、コーティング、半導体、モータなど供給者依存度を見る。

どの差分が品質保証で効くか

新しい機能ほど、評価法・基準・市場再現条件の定義が必要になる。
