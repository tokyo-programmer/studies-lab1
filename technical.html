2. 住宅設備機器の共通システム理解

5商品は一見別物に見えるが、実際には空気・熱・水・燃焼・電気・制御の組み合わせとして理解できる。 市場差は、この組み合わせのどこに重みが乗るかの違いである。

🌬 空気

レンジフードの換気、食洗機の乾燥、給湯器の排気、冷却ファンなど、多くの商品で基本になる。

🔥 熱

加熱・燃焼・断熱・熱交換・放熱が全商品で重要。熱を「作る・移す・逃がす」で整理する。

3–5. レンジフード

流体 / 油煙捕集 / モータ / 騒音

3. 基礎原理

レンジフードは、調理中に発生する油煙・水蒸気・臭気・微粒子を 捕集 → 搬送 → 排気 / 浄化の3段階で処理する機器である。 性能は単純な風量だけでは決まらず、開口形状、整流板、ダクト抵抗、フィルタ圧損、 調理上昇流との位置関係によって大きく変わる。

相似則：Q ∝ n　/ 　ΔP ∝ n²　/ 　P ∝ n³
システム圧損：ΔP_system = ΔP_filter + ΔP_duct + ΔP_bend + ΔP_outdoor
捕集：コンロ上の上昇流をどれだけ取り逃がさずフード内に導けるか。整流板や吸込口形状が効く。
搬送：ファン、ケーシング、流路、ダクトで圧損に打ち勝って油煙を移送する。高層住宅では高静圧が重要。
排気：屋外へ直接排気する方式。性能は高いが、ダクト工事・逆流防止・共同ダクト配慮が必要。
再循環：フィルタと活性炭で浄化して室内に戻す。HDBや賃貸、外壁制約のある市場で成立しやすい。

6–8. ガスコンロ / ビルトインガスホブ

燃焼 / 温調 / 安全 / 天板材料

6. 基礎原理

ガスコンロは、ガスと空気を適切に混合し、安定した炎として燃焼させ、 鍋底へ効率よく熱を伝える機器である。技術の中心は、 火力・熱効率・炎口設計・安全機構・清掃性にある。

FFD（立消え安全装置）：熱電対起電力 約20mV を利用し、炎消失時にガスを遮断
燃焼：一次空気・二次空気の取り方、炎口形状、熱流束、吹き消え耐性が設計の基本。
火力制御：高火力訴求だけでなく、煮込みや保温のような低火力安定制御も重要になる。
安全機構：FFD、過熱防止、鍋離れ検知、タイマ、自動消火など。市場によって訴求強度が異なる。
材料：黒ガラス、ステンレス、鋳鉄五徳、真鍮バーナーなど、意匠・耐久・清掃性を左右する。

9–11. IH / 電気ホブ

電磁誘導 / 制御 / センサ / UI

9. 基礎原理

IHは、コイルに高周波電流を流して磁束を発生させ、 鍋底金属内に渦電流を生じさせて加熱する方式である。 強みは、温度応答性、制御精度、安全性、UI連携にある。

高周波帯の目安：20〜50kHz
主要構成：加熱コイル / IGBT or MOSFET / PFC回路 / 冷却系 / センサ / UI
加熱原理：鍋自身を発熱体にするため、熱応答が速く、残熱制御もしやすい。
出力制御：センサ・インバータ制御により、微妙な低火力から高出力まで再現しやすい。
安全：チャイルドロック、鍋検知、過熱防止、残熱表示などで住宅向け安全性が高い。
連携：Bluetooth hood link、Home Connect など、他機器連動との親和性が高い。

12–14. 給湯器

熱交換 / 瞬間加熱 / 断熱 / 防食

12. 基礎原理

給湯器は、水へ必要熱量を与えて設定温度まで上げる機器であり、 市場ごとに instant / storage / hybrid / solar の比率が異なる。 技術の中心は、加熱方式、容量設計、設置制約、防食、断熱、安全にある。

瞬間式

必要時に通水しながら加熱

貯湯式

タンクに蓄えた温水を断熱保持

ハイブリッド

ヒートポンプ＋ガス等の熱源を組み合わせ効率最大化

14. 機能分解

14-1 瞬間 vs 貯湯：瞬間式は省スペース・連続出湯、貯湯式は多点給湯・ピーク対応に強い。
14-2 タンク構造：エナメル、SUS、断熱材、Mgアノード棒などで耐久性と腐食寿命が変わる。
14-3 省エネ：ヒートポンプ、断熱強化、制御最適化、ハイブリッド方式で効率を引き上げる。
14-4 衛生・保証：抗菌加熱、タンク衛生、長期保証は商品価値として見られやすい。

15–17. 食洗機

洗浄 / 乾燥 / 流体 / 静音 / 収納

15. 基礎原理

食洗機は、洗浄 → すすぎ → 乾燥の工程を制御し、 水・熱・洗剤・流体噴射・乾燥方式を組み合わせて食器を自動洗浄する機器である。 技術の中心は、洗浄性能、乾燥性能、静音、省スペース、使い勝手にある。

洗浄：ポンプ圧、スプレーアーム、温水、洗剤量、濁度センサで洗浄品質が決まる。
乾燥：余熱乾燥、ヒーター乾燥、Zeolith、DesertDry+ など方式差が大きい。
容量・サイズ：45cm、60cm、compact、countertop、portable の成立条件が市場で異なる。
静音・衛生：都市型住宅では静音が重要。高温除菌やサニタイズも訴求点になる。

17-2 乾燥技術：Zeolith などの吸放湿材料は、消費電力低減と乾燥性能向上を両立しやすい。
17-3 収納性：3rd rack、ラック可変、卓上・Drawer 型で、小空間でも使い勝手を高める。
17-4 小型成立条件：卓上型は設置簡便性が強みだが、容量・乾燥・給排水の制約をどう受け入れるかが鍵。

18. 商品横断の材料・構造・UI比較

黒ガラス：ガスホブ、IH、操作パネルで高級感・清掃性を両立しやすい。割れ、熱応力、指紋汚れ対策が必要。
SUS / アルミ / 鋼板：レンジフード外装や内部部材で使い分ける。SUSは耐食、鋼板はコスト、アルミは軽量性が利点。
エナメル / コーティング：給湯器タンク、五徳、内面部材などで耐食・清掃性・寿命に効く。親水/撥油/防汚が差別化要素。
静音化構造：BLDC化、防振ゴム、流路最適化、断熱吸音材、ポンプ防振などで実現する。都市市場では価値が高い。
清掃性設計：着脱しやすさ、フラット面、角の少なさ、コーティング、排液しやすい形状が共通論点。
タッチUI：高級感を出しやすいが、水・油・蒸気・手袋操作・誤動作への配慮が必須。メカスイッチとの使い分けも重要。

20. 技術編まとめ

20-1 競合製品をどう読むか

機能名だけでなく、その背景にある市場要求、住宅制約、施工条件、認証条件まで含めて読むことで、技術の意味が見える。

20-2 自社設計へどう翻訳するか

市場要求を、性能要件、構造要件、材料要件、制御要件、品質要件へ分解し、優先度と差別化ポイントを明確にすることが重要である。

20-3 次分冊への接続

ここで整理した技術差分を、製造・品質編で量産難易度、工程管理、検査、品質保証へつなげる。
